`softclassval` is released under [GNU GPL 3](http://www.gnu.org/licenses/gpl-3.0.html).

You can check the license of an R package typing:

{% highlight rconsole %}
>  packageDescription ("softclassval")$License
[1] "GPL"
{% endhighlight %}

{% highlight rconsole %}
> RShowDoc("GPL-3")
{% endhighlight %}

displays the text of the license.
